Detailed Description

Base class of the Voraus Framework.

This class provides functions to read and set the name parameter of a node and a function which register a module at the module manager. It also start a ros::Timer as main loop with a previous read parameter.

Set the node and name variables. Checks if a valid name and namespace was set. Otherwise it calls getNameStrings(). Also a loop time will be read from the parameter server and used to start the mainLoopCallback()
